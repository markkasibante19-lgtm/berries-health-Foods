 About The Project

This is a complete restaurant website built for **Berries Healthy Meals** – a healthy restaurant offering all-you-can-eat, happy-hour food, and vegan options in Kitende, Kampala.

### Key Features:
Hero Image Slideshow** – Auto-rotating food images with smooth transition
Mobile Responsive** – Works perfectly on phones, tablets, and desktops
WhatsApp Integration** – Reservation form sends directly to restaurant WhatsApp
Google Maps** – Interactive map showing exact location
Full Menu Page** – Organized menu with prices (Breakfast, Mains, Juices)
Reservation Form** – Book tables directly from the website
Customer Reviews Section** – Display testimonials with star ratings
Quick Action Buttons** – Call, Directions, WhatsApp – one tap on mobile

 How to Use

 Option 1: View Locally
1. Download the `index.html` file
2. Double-click to open in any web browser

Option 2: Host on GitHub Pages (Free)
1. Upload `index.html` to a GitHub repository
2. Go to **Settings → Pages**
3. Select branch: `main` → Save
4. Your site will be live at: `https://[username].github.io/[repository-name]/`

 Option 3: Host on Netlify (Free)
1. Drag and drop the `index.html` file to `netlify.com/drop`
2. Get an instant live link
